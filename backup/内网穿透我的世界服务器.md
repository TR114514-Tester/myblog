# 内网穿透是什么
内网穿透，也称为NAT穿透，是一种网络技术，允许外网用户访问位于内网（局域网）中的服务。简而言之，内网穿透使得没有公网IP地址的内网设备能够被外网访问，就像这些设备部署在公网服务器上一样
**但我相信你肯定不会了解这些东西，So**

使用内网穿透穿透我的世界服务器非常简单
先注册你的账号，登录上去后下载软件
有GUI登录后创建穿透
没GUI比如ChmlFrp直接在网页添加

| 类型  | 值  |
| ---- | ---- |
| 本地地址  | 127.0.0.1  |
| 本地端口  | 25565  |
| 穿透类型  | TCP  |
| 外网端口  | 0-65535随便填  |
| 名字  | 随便  |

创建完后开启穿透即可
推荐使用ChmlFrp
创建完隧道后找到配置文件
选择节点和自己的配置
生成一下就可以穿透了
右边会有Windows和Linux的启动脚本
```bash
# Windows启动脚本
frpc.exe -u XXXXXXXXXXXXXXXXXXX -p XXXXXX
# Linux启动脚本
curl -O https://www.chmlfrp.net/script/linux/frpc_install.sh && chmod +x frpc_install.sh && sudo ./frpc_install.sh "XXXXXXXXXXXXXXXXXXXXXXX" "圣何塞"
```

# 最后
最后把穿透出来的地址发给你的朋友，保证服务器和内网穿透都开着就可以进服务器了
例如`sh-2.frp.one:11451`

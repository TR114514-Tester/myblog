# 使用
打开[https://blog-assets.traveler.dpdns.org/assets/Theme.js](https://blog-assets.traveler.dpdns.org/assets/Theme.js)
复制里面的全部代码
放到你仓库的`static`目录下
点击`Actions > Build Gmeek > Run Workflow > RunWorkflow`
这样子就可以通过`yourusername.github.io/Theme.js`获取到主题内容了
# 配置
```javascript
    // ==================== 背景设置 ====================
    // 在这里设置你的背景图片地址
    const BACKGROUND_IMAGE_URL = 'https://blog-assets.traveler.dpdns.org/assets/image/background.png';
    
    // 可选：设置背景透明度（0-1之间）
    const BODY_BG_OPACITY = 0.8;
    const SIDENAV_BG_OPACITY = 0.6;
```
虽然有注释，但是我还是得讲一下

| 变量名  | 变量值  |
| ------------ | ------------ |
| BACKGROUND_IMAGE_URL  | 背景图片  |
| BODY_BG_OPACITY  | body的透明度  |
| SIDENAV_BG_OPACITY | sidenav的透明度  |
